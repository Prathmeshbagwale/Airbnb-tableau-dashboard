## 📌 Project Overview

An **end-to-end exploratory data analysis and interactive Tableau dashboard** built on real AirBnB listing data to uncover pricing trends, location performance, and revenue opportunities — simulating a real-world hospitality analytics use case.

---

## 📦 Dataset

| Detail | Info |
|--------|------|
| **Dataset** | AirBnB Seattle Listings Dataset |
| **Source** | [Download from Kaggle →](https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset) |
| **Alternate Source** | [Inside AirBnB — Seattle →](http://insideairbnb.com/get-the-data/) |
| **Key Columns** | Price, Neighbourhood, Room Type, Availability, Number of Reviews, Host ID |

> 💡 The dataset is too large to upload directly — download from Kaggle using the link above. Free Kaggle account required.

---

## ❓ Business Questions Answered

- Which **neighbourhoods and zip codes** command the highest average prices?
- How does **pricing vary by room type** (entire home vs private room)?
- What is the **estimated revenue potential** across different listing categories?
- How do **review scores** correlate with listing performance?
- What are the **seasonal availability trends** across the city?

---

## 🔍 Key Insights

- 📍 Location is the **#1 driver of price** — zip code analysis reveals up to 3x price differences across areas
- 🏡 Entire home listings generate **significantly higher revenue** than private or shared rooms
- 📅 Availability peaks align with **tourist season trends**
- ⭐ Review count alone doesn't determine price — listing quality and location dominate

## 🚀 How to View

**Option 1 — No install needed (recommended):**
👉 [Click here to open the live dashboard](https://public.tableau.com/views/AirBnB_Project_17777311740160/Dashboard1)

**Option 2 — Open locally in Tableau:**
1. Download the dataset from [Kaggle](https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset)
2. Download [Tableau Public](https://public.tableau.com/en-us/s/download) (free)
3. Clone this repo:
   ```bash
   git clone https://github.com/prathmesh/airbnb-tableau-dashboard.git
   ```
4. Open the `.twbx` file and re-connect to your downloaded dataset if prompted
